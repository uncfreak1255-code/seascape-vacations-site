# Monthly Reset

Run on the 1st of each month to set up fresh coordination files.

## Steps

1. **Create new task log**: `task-log-YYYY-MM.md` with header and empty structure
2. **Create new content priorities**: `content-priorities-YYYY-MM.md`
   - Check seasonality: What's the peak booking theme for this month?
   - Pull top keyword targets from Phase 3 plan that haven't been created yet
   - Set format rotation starting point based on last month's ending format
3. **Review previous month's task log**: Summarize wins, misses, and carry-forward items
4. **Update MEMORY.md**: Bump status on any completed phases, add new learnings

## Seasonal Themes

| Month | Theme | Peak Search Terms |
|-------|-------|-------------------|
| Jan-Feb | Winter escape, snowbird season | "florida winter vacation rentals" |
| Mar-Apr | Spring break, Easter | "spring break rentals gulf coast" |
| May-Jun | Summer kickoff, fishing season | "summer vacation rentals bradenton" |
| Jul-Aug | Peak summer, family vacations | "beach house rental anna maria island" |
| Sep-Oct | Shoulder season, fall deals | "affordable gulf coast rentals fall" |
| Nov-Dec | Holidays, Thanksgiving, Christmas | "christmas vacation rental florida" |

## Output

Confirm: "Monthly reset complete. New files created for [MONTH YEAR]. Top 3 content priorities: [list them]."
