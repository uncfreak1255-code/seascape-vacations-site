## Detailed Target Profiles

### 1. Anna Maria Island Chamber of Commerce

- **URL:** https://annamariaislandchamber.org/
- **Directory page:** https://annamariaislandchamber.org/directory/business-category/vacation-rentals/
- **Contact:** (941) 778-1541 | 5313 Gulf Dr, Holmes Beach, FL 34217
- **Link type:** Membership directory listing (dofollow business profile page)
- **Difficulty:** Easy — pay membership fee, get listed
- **Why it matters:** Direct relevance. Chamber directory pages rank for "Anna Maria Island vacation rentals" and related queries. Multiple chamber members show up in local pack results. The vacation rentals directory page is exactly where Seascape belongs.
- **Cost estimate:** $200-500/year for membership
- **Action:** Call or visit. Ask about membership tiers — some include featured listings with links.
